# 🚀 دليل البدء السريع - مشوار في Replit

## 📌 ابدأ هنا!

### الخطوة 1: إنشاء حساب Replit

1. اذهب إلى [replit.com](https://replit.com)
2. سجل بحسابك
3. اشترك في **Replit Core** ($20/شهر) للحصول على Replit Agent

### الخطوة 2: إنشاء المشروع

1. اضغط **"+ Create Repl"**
2. اختر **"Agent"** (أو **Node.js** template)
3. سمي المشروع: `mishwar-platform`

### الخطوة 3: استخدام البرومبتات

افتح الـ Agent وألصق البرومبتات بالترتيب من ملف `REPLIT_PROMPTS_READY.md`.

**الترتيب المهم:**

```
البرومبت 1 → انتظر الانتهاء → اختبر
البرومبت 2 → انتظر الانتهاء → اختبر
البرومبت 3 → انتظر الانتهاء → اختبر
... وهكذا
```

⚠️ **لا تعطي الـ Agent برومبتين مرة واحدة**

---

## 💡 نصائح ذهبية

### 1. خذ وقتك
كل برومبت يستغرق 15-30 دقيقة. لا تستعجل.

### 2. احفظ التقدم
بعد كل برومبت ناجح، اعمل Git commit:

```bash
git add .
git commit -m "Complete prompt N: description"
```

### 3. اختبر باستمرار

بعد كل برومبت، شغل:

```bash
# للـ Backend
cd apps/api && npm run dev

# للـ Frontend
cd apps/web && npm run dev
```

### 4. إذا حدث خطأ

استخدم:

```
Fix all TypeScript errors in the project.
```

أو:

```
The build is failing with this error: [paste error]
Please fix it.
```

### 5. لا تفتح كل الـ Apps مرة واحدة

Replit له موارد محدودة. شغل تطبيق واحد في كل مرة:
- API على port 3000
- Web على port 3001
- Admin على port 3002
- Driver Portal على port 3003

## 🆘 المشاكل الشائعة وحلولها

### مشكلة: Replit Agent يفتح Project جديد

**الحل:** اطلب منه:
```
Continue working in the same Repl, don't create a new one.
```

### مشكلة: Out of memory

**الحل:** 
1. أوقف بعض الـ apps
2. اطلب: `Clear unnecessary files and restart`

### مشكلة: PostgreSQL لا يعمل

**الحل:** في Replit:
1. اذهب لـ **Tools** → **Database**
2. أضف PostgreSQL database
3. سيضع DATABASE_URL تلقائياً في secrets

### مشكلة: Mapbox لا يظهر

**الحل:** 
1. سجل في [mapbox.com](https://mapbox.com) (مجاني)
2. خذ public token
3. ضعه في `.env`:
   ```
   NEXT_PUBLIC_MAPBOX_TOKEN=pk.eyJ1...
   ```

### مشكلة: الترجمة لا تعمل

**الحل:** اطلب:
```
Check next-intl configuration and make sure ar.json and en.json are loaded correctly.
```

### مشكلة: RTL لا يعمل

**الحل:** اطلب:
```
Make sure dir="rtl" is set on html tag for Arabic, and Tailwind RTL plugin is configured.
```
