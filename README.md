# مشوار

منصة نقل ذكية شاملة للسوق السعودي والسوداني

## Architecture

- **Landing & Riders App** - Flagship rider experience
- **Drivers Portal** - Driver dashboard and management
- **Car Owners Portal** - Vehicle owner fleet management
- **Business Portal** - Corporate transportation accounts
- **Admin Panel** - Platform administration

## Stack

- Express 5 + Drizzle ORM (PostgreSQL)
- React + Vite + Wouter + TanStack Query + shadcn/ui
- pnpm monorepo workspace

## Development

Managed through Replit workflows. See artifacts in `artifacts/` directory.
